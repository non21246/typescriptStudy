# Registration Form
Features:

- Environment, Typescript, Nodemon setup
- MongoDB & Mongoose connect, Database creation
- Controllers creation
- Middlewares creation
- Cookie based authentication
- Postman testing
- Create, Read, Update

### Cloning the repository

```shell
git clone 'url'
```

### Install packages

```shell
npm i
```

### Setup MongoDB URL

In `src/index.ts`:

```shell
npm start
```
